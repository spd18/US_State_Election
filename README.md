# US Senate Election Data Analysis Dashboard

## Project Overview
This project analyzes **U.S. Senate election data from 1976 to 2020** to understand political trends, voting behavior, and party dominance across different states in the United States.

The objective of the project is to transform raw election data into meaningful insights using **data preprocessing, exploratory data analysis (EDA), and interactive dashboards**.

Using **Python for data preprocessing and Tableau for visualization**, we developed multiple dashboards that illustrate election outcomes, vote share distribution, and party power dynamics over time.

The dashboards are integrated into a **web application architecture** using **React (frontend), Django (backend), and PostgreSQL (database)** to allow users to explore the data interactively.

---

## Dataset Source
The dataset used in this project comes from **Harvard Dataverse** and contains U.S. Senate election results from **1976–2020** at the state level.

Dataset includes:
- Election year
- State
- Candidate names
- Party affiliation
- Total votes
- Candidate vote counts
- Election results

Dataset Link:  
https://dataverse.harvard.edu/file.xhtml?fileId=7609736&version=7.0&toolType=PREVIEW

## Tableau Dashboards
The project includes four interactive Tableau dashboards.

### 1. Party Power Dynamics Over Time
Shows the evolution of party dominance and vote share trends across election years.  
https://public.tableau.com/app/profile/athul.narayanan2251/viz/ELECTIONVIEW1/Dashboard1

### 2. State-Level Election Winners
Visualizes election winners by state along with vote share distribution.  
https://public.tableau.com/app/profile/athul.narayanan2251/viz/ELECTIONDASHBOARD2/WinnerandVoteShare

### 3. Special Election Trends
Displays trends in special elections including participating states, vote share, and election outcomes.  
https://public.tableau.com/app/profile/athul.narayanan2251/viz/Election3_17445537635860/Dashboard1

### 4. Party Performance
Shows the overall performance of political parties across multiple election years.  
https://public.tableau.com/app/profile/parth.sathiya/viz/test_17418809958310/Dashboard2
